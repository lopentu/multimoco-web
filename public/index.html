臺大多模態語料庫

Multimodal Corpus for Formosan Languages

關於 About

This project aims to serve as the pioneering work on the construction of a large-scale multimodal corpus for Formosan Languages (MultiMoCo) including four official language groups in Taiwan, as well as the corpus analysis tools enhanced by both human annotation and recent multimodal machine learning techniques. Throughout this long-term project, our primary goal is set to provide an empirical base to support/evaluate the cognitive linguistic theoretical claim.
